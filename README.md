# Jueves

App del grupo de los muchachos del jueves.

## Tecnología
- React 18 + Vite
- Supabase (base de datos)
- Tailwind CSS
- PWA instalable

## Variables de entorno (Vercel)

```
VITE_SUPABASE_URL=https://...supabase.co
VITE_SUPABASE_KEY=sb_publishable_...
VITE_GROUP_PASSWORD=...
VITE_ADMIN_PASSWORD=...
```
